Tech Stack

기술 스택: 빠른 프로토타입과 검증 루프 중심

  • Upload · Recipe Builder · Generate & Audit UI

AI / Tools

  • OpenAI: facet analysis · coherence judge · audit
  • v0: React + Tailwind code generation
  • sharp: pixel-based color token extraction

Endpoint역할결과
